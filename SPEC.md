# NEON MATCH - Спецификация проекта

## 1. Обзор
- **Название:** NEON MATCH
- **Жанр:** Match-3 головоломка с мета-игрой
- **Стиль:** Neon Cyberpunk
- **Платформа:** Web (браузер)
- **Технологии:** Phaser.js 3, TypeScript, Vite

## 2. Визуальный стиль

### Палитра
| Цвет | HEX | Использование |
|------|-----|--------------|
| Neon Pink | #FF00FF | Основной акцент |
| Cyan | #00FFFF | Вторичный акцент |
| Electric Blue | #0066FF | Третичный |
| Dark BG | #0A0A0F | Фон |
| Grid | #1A1A2E | Сетка поля |

### Плитки (6 типов)
1. **Cyan Crystal** (#00FFFF) - hexagon shape
2. **Pink Crystal** (#FF00FF) - hexagon shape  
3. **Blue Crystal** (#0066FF) - hexagon shape
4. **Green Crystal** (#00FF66) - hexagon shape
5. **Yellow Crystal** (#FFFF00) - hexagon shape
6. **Purple Crystal** (#9900FF) - hexagon shape

### Бонусы
- **Rocket (4 match):** Стрелка, очищает ряд/колонку
- **Bomb (L/T match):** Взрыв 3x3
- **Color Bomb (5 match):** Удаляет все одного цвета

### UI
- Шрифт: Orbitron
- Glow-эффекты на кнопках
- Глитч-анимации при переходах

### Анимации (время в мс)
- Swap: 250ms
- Destroy: 150ms
- Fall: 200ms
- Bonus activation: 300ms

## 4. Механики

### Базовая логика
1. Поле 8x8
2. Свайп/клик для обмена соседних плиток
3. Поиск совпадений 3+ в ряд
4. Удаление, гравитация, респавн
5. Проверка "возможных ходов"

### Бонусы
| Комбинация | Бонус | Эффект |
|------------|-------|--------|
| 4 в ряд | Rocket H | Очищает горизонталь |
| 4 в ряд | Rocket V | Очищает вертикаль |
| L-форма | Bomb | Очищает 3x3 |
| T-форма | Bomb | Очищает 3x3 |
| 5 в ряд | Color Bomb | Удаляет цвет |
| Bonus + Bonus | Combo | Усиленный |

### Мета-игра
- **Энергия:** 30 единиц, восстанавливается по 1 за 5 минут
- **Монеты:** Зарабатываются за уровни
- **Звёзды:** За выполнение целей
- **Комната:** Декор за звёзды

## 5. Уровни

### Генерация
Процедурная генерация с:
- Целями (очки, собрать элементы)
- Ограничением ходов
- Нарастающей сложностью

### Цели уровня
- Набрать X очков
- Собрать X элементов определённого цвета
- Уничтожить X препятствий

## 6. UI Экраны

### MainMenu
- Логотип с глитч-анимацией
- Кнопка "PLAY"
- Кнопка "ROOM" (мета-игра)
- Текущие ресурсы (монеты, звёзды, энергия)

### Game
- Поле 8x8 по центру
- HUD: ходы, цель, бонусы
- Кнопка паузы

### Room (Meta)
- Вид комнаты
- Предметы для покупки
- Кнопка назад

## 7. Технические требования

### Поддержка устройств
- Desktop: мышь
- Mobile: touch events
- Разрешение: адаптивное (fit to screen)

### Производительность
- 60 FPS
- < 3MB bundle size
- Асинхронная загрузка

## 8. Психологические механики (из анализа)

### Juiciness
- Вспышки при матчах
- Частицы при уничтожении
- Звуковые эффекты
- Screen shake при бонусах

### DDA (Dynamic Difficulty)
- Если игрок застрял → подсказка
- Если проходит слишком легко → сложность+
- "Near-miss" эффект

### FOMO
- Ограниченные события
- Ежедневные награды
